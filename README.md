# UD05A05 - sentencias iterativas - bucle while

## Ejercicio 1
```
Este código cuenta del 1 al 10 utilizando un bucle while y muestra cada número en la consola.
```

## Ejercicio 2
```
Este código cuenta del 1 al 10 e imprime cada número en una línea, utilizando un bucle do-while.
```

## Ejercicio 3
```
Este código es una función llamada countTo que toma un parámetro llamado hasta. Convierte hasta en un número y verifica si es un número válido. Si es así, imprime números del 1 hasta el número dado en orden ascendente. Si hasta no es un número válido, muestra un mensaje de error indicando que el parámetro debe ser un número.
```

## Ejercicio 4
```
Este código es una función llamada countTo que cuenta desde 1 hasta el número especificado en el parámetro 'hasta'. Si 'hasta' no es un número, muestra un mensaje de error. Luego, la función se llama con varios ejemplos para contar hasta diferentes números y para un valor no numérico.
```

## Ejercicio 5
```
Este código en JavaScript define una función llamada countFromTo que cuenta desde un número dado (desde) hasta otro número dado (hasta). Primero, convierte los argumentos desde y hasta en números. Luego, comprueba si ambos son números. Si son números y desde es menor que hasta, imprime cada número desde desde hasta hasta usando un bucle while. Si desde no es menor que hasta, muestra un mensaje de error indicando que desde debe ser menor que hasta. Si alguno de los argumentos no es un número, muestra un mensaje de error indicando que ambos parámetros deben ser números. Finalmente, la función se invoca cuatro veces con diferentes argumentos para demostrar su funcionamiento.
```

## Ejercicio 6
```
Este código es una función llamada countFromTo que toma dos parámetros, desde y hasta, y cuenta desde el valor desde hasta el valor hasta. Se asegura de que ambos parámetros sean números y que desde sea menor que hasta. Si los parámetros son válidos, cuenta hacia arriba desde desde hasta hasta, imprimiendo cada número en la consola. Si desde es mayor o igual que hasta, muestra un mensaje de error indicando que desde debe ser menor que hasta. Si uno o ambos de los parámetros no son números, muestra un mensaje de error indicando que ambos parámetros deben ser números.
```
